<p>한국투자증권에서 제공하는 웹 소켓과 API를 사용했어요.</p>

GitHub 저장소를 clone하고 stock 디렉토리에서 터미널을 열고 아래 명령어를 차례로 입력해주세요.

```
$ npm i
$ npm start
```

어플리케이션은 [http://localhost:3000](http://localhost:3000)에서 실행돼요.
